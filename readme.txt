=== Crocoblock Addons ===
Requires at least: 5.0
Tested up to: 6.7.1
Stable tag: 1.0.1

== Description ==
Crocoblock Addons is a powerful plugin that enhances your WordPress site with additional features and functionalities. This plugin is designed to work seamlessly with Crocoblock Jet Plugins.

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/crocoblock-addons` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the Crocoblock Addons settings page to configure the plugin.
